# Python Developer Roadmap - Code Curriculum

Welcome to your **Code-First** Python learning journey!
This curriculum is designed to teach you Python by reading and running actual code.

## How to Use
1.  Navigate to a folder (e.g., `01_Basics`).
2.  Open a file (e.g., `01_variables_and_data_types.py`) in your code editor.
3.  Read the comments to understand the concept.
4.  Run the file to see the output: `python 01_Basics/01_variables_and_data_types.py`
